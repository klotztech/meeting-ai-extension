## Step 1: Create Icons (2 minutes)

**Option A: Use the Icon Generator**

1. Open `create-icons.html` in your browser
2. Customize the icon (emoji/text and color)
3. Click "Download All Icons"
4. Save the three PNG files to the `icons/` folder

**Option B: Quick Placeholder**

- Create any three PNG images named `icon16.png`, `icon48.png`, `icon128.png`

## Step 2: Load Extension (1 minute)

1. Open Chrome
2. Go to `chrome://extensions/`
3. Enable **Developer mode** (top-right toggle)
4. Click **Load unpacked**
5. Select the `meeting-ai-extension` folder
6. ‚úÖ Extension loaded!

## Step 3: Test It (2 minutes)

1. Open any YouTube video (or a meeting)
2. Click the extension icon in Chrome toolbar
3. Click **"Start Recording"**
4. Allow microphone access if prompted
5. Let it record for 10-20 seconds
6. Click **"Stop Recording"**

## Troubleshooting

**Extension icon not showing?**

- Make sure icons are in the `icons/` folder
- Check `chrome://extensions/` for errors

**Recording not working?**

- Grant microphone permissions
- Make sure you're on a page with audio
- Try refreshing the page
